**tamazight.education** is a learning platform for the Tamazight language. Lessons are interactive, access is free, and everything is built around a simple content model: a teacher creates, a learner progresses.

The platform is currently in its first proof of concept — validating the core learning loop before building anything larger.

### Pedagogy

Pedagogical content and didactic design are led by **Essaadia Hamouch**, Tamazight teacher trainee at CRMEF Guelmim.
